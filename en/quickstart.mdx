To begin using nQivi Studio, follow these four simple steps to set up your account, connect the software, and render your first video.

---

## 4 Steps to Get Started

### Step 1: Create an Account & Purchase Credits
Visit [studio.nqivi.com](https://studio.nqivi.com) and register a new account. Credits are the internal currency of the platform, used to pay for automated AI tasks (like script writing, translation, voice dubbing, or rendering).
* Go to the **Pricing** or **Billing** page to add credits.
* The system supports flexible pay-as-you-go top-ups starting at **20,000 VND** or **$2 USD**.

### Step 2: Download & Install the Client App
Go to the **Download** tab on the website and download the nQivi Studio installer for your operating system.
* Ensure you also install the required rendering dependencies (ImageMagick and FFmpeg) on your machine.
* For step-by-step setup details, refer to the [Installation Guide](/en/installation).

### Step 3: Generate and Link your API Key
The Desktop Client requires an API key to communicate securely with the nQivi servers and authorize credit consumption.
1. Navigate to the **Api Keys** page on the Website Portal.
2. Click **Create API Key**, then copy the generated key string.
3. Open the nQivi Client Desktop App, navigate to the **Settings** tab, and paste the key into the **SERVER_API_KEY** input field.
4. Click **Save** to activate the software.

### Step 4: Start Creating Content!
Navigate to the **Studio** tab inside the Client App:
* Use **Autopilot** to generate a script via AI prompts, select a high-quality voice, choose video/image stock assets, configure subtitle fonts, and click **Start Task**.
* Track your rendering job in the **Task Manager** tab. Once completed, your finished video will be available in the output directory and the **Content Library**!
